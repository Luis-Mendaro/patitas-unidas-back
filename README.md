# Patitas Unidas — Backend API

API REST para la plataforma de adopción de mascotas Patitas Unidas.

**API:** https://patitas-unidas-back-2026.vercel.app

---

## Tecnologías

- Node.js + Express
- Sequelize ORM
- PostgreSQL (Supabase)
- JWT para autenticación
- Supabase Storage para imágenes
- Desplegado en Vercel

---

## Endpoints principales

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | /auth/login | Login de usuarios y refugios |
| GET | /pets | Listado de mascotas con filtros |
| GET | /shelters | Listado de refugios |
| POST | /users | Registro de usuario |
| GET | /requests | Solicitudes de adopción |
